Flask

Flask Install

Flask Update/Upgrade

pip3 install --upgrade flask

Flask Run App

flask run --host=0.0.0.0 --port=5080

Host all zeros allow the site to be viewable on localhost and external IP Address

Virtual Environment Server

Setup Virtual Environment

Good practice to have a Virtual Environment running your python and flask, ensure the virtual environment is running before installing Flask - ie install Flask within the virtual environment

python3 -m venv [Name]

[Name] - replace with your preferred Environment name

Activate Virtual Environment

source [name]/bin/activate

Check what packages and versions are running within V-env

pip list
